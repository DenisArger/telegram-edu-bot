2. SETUP_AIRTABLE.md - настройка Airtable
3. REGISTER_STUDENT.md - регистрация первого ученика

1. task.md - техническое задание
2. architecture.md - понимание архитектуры

2. SECURITY.md - настройка безопасности
3. GIT_SETUP.md - настройка Git (если нужно)

1. REGISTER_STUDENT.md - проблемы с регистрацией
2. SETUP_AIRTABLE.md - проблемы с Airtable
3. PUSH_FIX.md - проблемы с GitHub
4. GITHUB_AUTH.md - проблемы с аутентификацией
