# Руководство для учителя: Создание заданий в Airtable

Это руководство поможет учителю создавать и управлять заданиями для учеников через Airtable без необходимости работы с кодом.

## Обзор

Учитель работает исключительно через интерфейс Airtable. Все задания, уроки и ученики управляются через таблицы в Airtable. Бот автоматически загружает данные и выдает задания ученикам в Telegram.

## Структура базы данных

В вашей базе Airtable должны быть следующие таблицы:

1. **Уроки** - список всех уроков
2. **Задания** - все задания для уроков
3. **Ученики** - список зарегистрированных учеников
4. **Прогресс** - история выполнения заданий (заполняется автоматически)

## Шаг 1: Создание урока

Перед созданием заданий нужно создать урок:

1. Откройте таблицу **"Уроки"** в Airtable
2. Нажмите кнопку **"Add a record"** или **"+"** для создания новой записи
3. Заполните поле **"Название"** или **"Имя"** (например: "Математика 5 класс", "Русский язык - Правописание")
4. Сохраните запись

**Важно**: Запомните или скопируйте ID урока - он понадобится при создании заданий.

## Шаг 2: Создание задания

### 2.1. Основные поля задания

1. Откройте таблицу **"Задания"** в Airtable
2. Нажмите **"Add a record"** для создания нового задания
3. Заполните обязательные поля:

#### Поле "Активно" (Checkbox)
- ✅ **Включите чекбокс** - задание будет видно ученикам
- ❌ **Выключите чекбокс** - задание скрыто от учеников (можно использовать для черновиков)

#### Поле "Урок" (Link to "Уроки")
- Нажмите на поле и выберите урок из списка
- Или начните вводить название урока для поиска
- **Важно**: Задание будет доступно только ученикам, у которых выбран этот урок

#### Поле "Текст задания" (Single line text)
- Введите текст задания, который увидит ученик
- Примеры:
  - "Решите уравнение: 2x + 5 = 15"
  - "Вставьте пропущенные буквы: к_р_ндаш"
  - "Назовите столицу Франции"

#### Поле "Правильные ответы" (Long text)
- Введите все правильные варианты ответов
- **Каждый ответ с новой строки**
- Пример:
  ```
  x=5
  5
  пять
  ```
- **Важно**: 
  - Ответы проверяются без учета регистра и пробелов
  - Если правильных вариантов несколько, введите каждый с новой строки
  - Пустые строки игнорируются

#### Поле "Комментарий (верно)" (Single line text)
- Сообщение, которое увидит ученик при правильном ответе
- Примеры:
  - "Правильно! ✅"
  - "Молодец! Переходим к следующему заданию."
  - "Верно! Это действительно столица Франции."

#### Поле "Комментарий (ошибка)" (Single line text)
- Сообщение, которое увидит ученик при неправильном ответе
- Примеры:
  - "Неправильно. Попробуйте еще раз."
  - "Неверно. Подсказка: подумайте о правописании."
  - "Ошибка. Попробуйте решить уравнение пошагово."

### 2.2. Опциональные поля

#### Поле "Класс" (Text или Number)
- Укажите класс, для которого предназначено задание (например: "5", "10", "11А")
- **Если поле пустое**: задание будет видно всем ученикам этого урока
- **Если указан класс**: задание будет видно только ученикам этого класса
- **Пример использования**: 
  - Задание для 5 класса: укажите "5"
  - Задание для всех: оставьте поле пустым

#### Поле "order" или "Порядок" (Number)
- Порядковый номер задания (1, 2, 3...)
- **Примечание**: В текущей версии задания показываются в случайном порядке, но поле можно заполнить для будущего использования

### 2.3. Пример заполненного задания

| Поле | Значение |
|------|----------|
| Активно | ✅ (включено) |
| Урок | Математика 5 класс |
| Текст задания | Решите уравнение: 2x + 5 = 15 |
| Правильные ответы | x=5<br>5<br>пять |
| Комментарий (верно) | Правильно! ✅ |
| Комментарий (ошибка) | Неправильно. Попробуйте еще раз. |
| Класс | 5 |
| Порядок | 1 |

## Шаг 3: Активация задания

После создания задания:

1. Убедитесь, что чекбокс **"Активно"** включен ✅
2. Проверьте, что поле **"Урок"** заполнено и указывает на нужный урок
3. Убедитесь, что все обязательные поля заполнены:
   - Текст задания
   - Правильные ответы
   - Комментарии (верно и ошибка)

**Важно**: Задание появится у учеников только после того, как:
- Чекбокс "Активно" включен
- Урок указан правильно
- Ученик зарегистрирован на этот урок

## Шаг 4: Редактирование задания

Чтобы изменить задание:

1. Откройте таблицу **"Задания"**
2. Найдите нужное задание
3. Измените нужные поля
4. Изменения сохраняются автоматически

**Примечание**: Если ученик уже начал урок, изменения применятся при следующем запуске урока.

## Шаг 5: Деактивация задания

Чтобы временно скрыть задание от учеников:

1. Откройте задание в таблице **"Задания"**
2. Выключите чекбокс **"Активно"** ❌
3. Задание больше не будет показываться ученикам

**Совет**: Не удаляйте задания! Лучше деактивировать их, чтобы сохранить историю и статистику.

## Шаг 6: Создание урока с несколькими заданиями

Для создания полноценного урока:

1. Создайте урок в таблице **"Уроки"** (см. Шаг 1)
2. Создайте несколько заданий в таблице **"Задания"**:
   - Каждое задание должно быть связано с этим уроком (поле "Урок")
   - Заполните все обязательные поля для каждого задания
   - Активируйте все задания (чекбокс "Активно")
3. Ученики увидят задания в случайном порядке

## Примеры заданий для разных предметов

### Математика

**Текст задания**: Решите уравнение: 3x - 7 = 14

**Правильные ответы**:
```
x=7
7
x = 7
```

**Комментарий (верно)**: Правильно! x = 7

**Комментарий (ошибка)**: Неправильно. Попробуйте перенести числа в одну сторону.

### Русский язык

**Текст задания**: Вставьте пропущенные буквы: к_р_ндаш

**Правильные ответы**:
```
карандаш
Карандаш
```

**Комментарий (верно)**: Правильно! ✅

**Комментарий (ошибка)**: Неправильно. Подсказка: это предмет для письма.

### География

**Текст задания**: Назовите столицу Франции

**Правильные ответы**:
```
Париж
париж
```

**Комментарий (верно)**: Верно! Столица Франции - Париж.

**Комментарий (ошибка)**: Неправильно. Попробуйте еще раз.

### История

**Текст задания**: В каком году началась Первая мировая война?

**Правильные ответы**:
```
1914
1914 год
```

**Комментарий (верно)**: Правильно! Первая мировая война началась в 1914 году.

**Комментарий (ошибка)**: Неправильно. Война началась в начале XX века.

## Фильтрация заданий по классу

Если нужно создать задания для конкретного класса:

1. Создайте задание как обычно
2. В поле **"Класс"** укажите номер класса (например: "5", "10", "11А")
3. Задание будет видно только ученикам этого класса

**Важно**: 
- Если у ученика не указан класс, он увидит только задания без указания класса
- Если у ученика указан класс "5", он увидит:
  - Все задания без указания класса (универсальные)
  - Все задания с классом "5"

## Проверка заданий

Чтобы проверить, какие задания видны ученикам:

1. Запустите тестовый скрипт (если у вас есть доступ к коду):
```bash
node src/check-tasks.js
```

2. Или проверьте вручную в Airtable:
   - Откройте таблицу "Задания"
   - Отфильтруйте по полю "Активно" = TRUE
   - Проверьте, что поле "Урок" заполнено
   - Проверьте, что все обязательные поля заполнены

## Частые ошибки и их решение

### Задание не появляется у ученика

**Причины**:
1. Чекбокс "Активно" выключен → включите его
2. Поле "Урок" не заполнено → выберите урок
3. Ученик не зарегистрирован на этот урок → назначьте урок ученику
4. Задание предназначено для другого класса → проверьте поле "Класс"

### Ученик видит неправильные задания

**Причины**:
1. У ученика указан неправильный урок → проверьте поле "Текущий урок" в таблице "Ученики"
2. Задания связаны с другим уроком → проверьте поле "Урок" в заданиях

### Ответ не засчитывается как правильный

**Причины**:
1. Ответ не указан в поле "Правильные ответы" → добавьте все возможные варианты
2. Лишние пробелы или другой регистр → проверьте, что ответы введены правильно (бот нормализует ответы автоматически)
3. Опечатка в правильных ответах → проверьте поле "Правильные ответы"

## Советы для учителя

1. **Создавайте уроки заранее**: Сначала создайте урок, затем задания для него
2. **Используйте черновики**: Создавайте задания с выключенным "Активно", затем активируйте когда будете готовы
3. **Множественные правильные ответы**: Добавляйте все возможные варианты ответов (с заглавной буквы, без, с пробелами и без)
4. **Полезные комментарии**: Напишите комментарии, которые помогут ученику понять ошибку или закрепить правильный ответ
5. **Фильтрация по классу**: Используйте поле "Класс" для создания заданий разной сложности
6. **Не удаляйте задания**: Деактивируйте их вместо удаления, чтобы сохранить статистику

## Просмотр прогресса учеников

Все ответы учеников автоматически сохраняются в таблице **"Прогресс"**:

- **Ученик** - ссылка на ученика
- **Задание** - ссылка на задание
- **Ответ ученика** - ответ, который дал ученик
- **Верно** - правильность ответа (чекбокс)

Используйте эту таблицу для:
- Анализа успеваемости
- Понимания, какие задания вызывают трудности
- Отслеживания прогресса учеников

## Дополнительная помощь

Если у вас возникли вопросы:

1. Проверьте структуру таблиц в разделе "Структура базы данных Airtable" в документации
2. Убедитесь, что все поля заполнены правильно
3. Проверьте, что задание активировано (чекбокс "Активно")
4. Убедитесь, что ученик зарегистрирован на нужный урок
