# Как добавить или изменить урок для ученика

Есть несколько способов добавить или изменить урок для ученика:

## Способ 1: Через бота (рекомендуется)

### Для ученика:
1. Отправьте команду `/start` в Telegram
2. Нажмите кнопку **"📚 Выбрать другой урок"**
3. Выберите нужный урок из списка
4. Нажмите **"▶️ Начать новый урок"**

### Для учителя (если нужно изменить урок ученика):
1. Откройте таблицу "Ученики" в Airtable
2. Найдите ученика по Telegram ID
3. Измените поле "Текущий урок" - выберите нужный урок из списка
4. Сохраните изменения

## Способ 2: Напрямую в Airtable

### Шаги:
1. Откройте вашу базу данных в Airtable
2. Перейдите в таблицу **"Ученики"**
3. Найдите нужного ученика (можно использовать фильтр по "Telegram ID")
4. В поле **"Текущий урок"**:
   - Если поле пустое - выберите урок из выпадающего списка
   - Если урок уже назначен - выберите другой урок из списка
5. Сохраните изменения

### Важно:
- Поле "Текущий урок" должно быть типа **Link to "Уроки"**
- Урок должен существовать в таблице "Уроки"
- После изменения урока в Airtable, ученик может использовать `/start` для начала нового урока

## Способ 3: Программно (для разработчиков)

Если нужно изменить урок программно, используйте функцию `updateStudentLesson()`:

```javascript
import { updateStudentLesson, getStudentByTelegramId } from "./student.js";

// Получить ученика
const student = await getStudentByTelegramId(telegramId);

// Изменить урок
await updateStudentLesson(student.id, newLessonId);
```

## Структура данных

В таблице "Ученики" должно быть:
- **Telegram ID** (Number) - ID пользователя в Telegram
- **Текущий урок** (Link to "Уроки") - связь с одним уроком

> **Примечание:** Сейчас бот поддерживает один текущий урок на ученика. Если нужно добавить несколько уроков одновременно, потребуется изменить структуру таблицы.

## Добавление нового урока в систему

Если нужно добавить новый урок в систему (не для конкретного ученика, а вообще):

1. Откройте таблицу **"Уроки"** в Airtable
2. Создайте новую запись
3. Заполните поле **"Название"** (или **"Имя"**)
4. Сохраните
5. Теперь этот урок будет доступен в боте при выборе урока

## Проверка доступных уроков

Чтобы проверить, какие уроки доступны:

```bash
cd telegram-rus-bot
node src/check-tasks.js
```

Этот скрипт покажет:
- Всех учеников и их текущие уроки
- Все доступные уроки
- Все задания для каждого урока
