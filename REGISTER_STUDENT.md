# Как зарегистрировать ученика в Airtable

Бот отвечает "Вы не зарегистрированы" потому, что ваш Telegram ID не найден в таблице "Ученики" в Airtable.

## Шаг 1: Узнайте свой Telegram ID

Есть несколько способов:

### Способ 1: Использовать бота @userinfobot
1. Откройте Telegram
2. Найдите бота [@userinfobot](https://t.me/userinfobot)
3. Отправьте ему `/start`
4. Он покажет ваш ID (например: `282850458`)

### Способ 2: Использовать бота @getidsbot
1. Найдите бота [@getidsbot](https://t.me/getidsbot)
2. Отправьте ему любое сообщение
3. Он покажет ваш ID

## Шаг 2: Добавьте запись в таблицу "Ученики" в Airtable

1. Откройте вашу базу данных в Airtable
2. Перейдите в таблицу **"Ученики"**
3. Создайте новую запись со следующими полями:

### Обязательные поля:

- **`Telegram ID`** (Number) - ваш Telegram ID (например: `282850458`)
  - ⚠️ **Важно:** Это должно быть число, а не текст!

- **`Текущий урок`** (Link to "Уроки") - выберите урок из связанной таблицы
  - Если у вас есть таблица "Уроки", выберите нужный урок
  - Если таблицы "Уроки" нет, создайте её или используйте ID урока напрямую

### Пример записи:

| Telegram ID | Текущий урок |
|-------------|--------------|
| 282850458   | [ссылка на урок] |

## Шаг 3: Проверьте структуру таблицы "Ученики"

Убедитесь, что в таблице "Ученики" есть следующие поля:

1. **Telegram ID** - тип поля: `Number` (не Text!)
2. **Текущий урок** - тип поля: `Link to another record` → связь с таблицей "Уроки"

## Шаг 4: Проверьте, что урок существует

Если вы используете связь с таблицей "Уроки":

1. Убедитесь, что таблица "Уроки" существует
2. Создайте хотя бы один урок в этой таблице
3. Используйте этот урок в поле "Текущий урок" ученика

## Шаг 5: Проверьте, что в уроке есть задания

В таблице "Задания" должны быть записи:

- **`Активно`** = ✅ (чекбокс включен)
- **`Урок`** = ссылка на ваш текущий урок
- **`Текст задания`** - текст задания
- **`Правильные ответы`** - правильные ответы (каждый с новой строки)
- **`Комментарий (верно)`** - комментарий при правильном ответе
- **`Комментарий (ошибка)`** - комментарий при неправильном ответе
- **`order`** или **`Порядок`** - порядковый номер задания

## Шаг 6: Протестируйте бота снова

После добавления записи:

1. Откройте чат с ботом в Telegram
2. Отправьте команду `/start`
3. Бот должен ответить: "Начинаем урок русского языка ✍️"
4. Затем бот отправит первое задание

## Устранение проблем

### Бот всё ещё говорит "Вы не зарегистрированы"

1. **Проверьте Telegram ID:**
   - Убедитесь, что ID в Airtable точно совпадает с вашим реальным ID
   - Проверьте, что поле имеет тип `Number`, а не `Text`

2. **Проверьте имя таблицы:**
   - Таблица должна называться точно **"Ученики"** (с заглавной буквы)
   - Проверьте, нет ли лишних пробелов в названии

3. **Проверьте права доступа:**
   - Убедитесь, что Personal Access Token имеет права на чтение таблицы "Ученики"

### Бот не отправляет задания

1. **Проверьте поле "Текущий урок":**
   - Убедитесь, что оно заполнено
   - Убедитесь, что урок существует

2. **Проверьте таблицу "Задания":**
   - Убедитесь, что есть задания с `Активно = TRUE()`
   - Убедитесь, что поле "Урок" в заданиях ссылается на ваш текущий урок

## Быстрая проверка через тестовый скрипт

Запустите тестовый скрипт, чтобы проверить подключение:

```bash
cd telegram-rus-bot
node src/test-airtable.js
```

Это покажет, доступны ли таблицы и есть ли в них данные.
