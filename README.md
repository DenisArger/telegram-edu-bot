# Telegram Bot для уроков русского языка

Telegram-бот для выдачи заданий по русскому языку с интеграцией Airtable.

## Описание

Бот предназначен для учеников и работает исключительно через Telegram. Учитель управляет данными через Airtable без необходимости работы с кодом.

## Возможности

- Выдача заданий по русскому языку
- Проверка ответов учеников
- Сохранение прогресса в Airtable
- Комментарии к правильным и неправильным ответам
- Управление уроками через Airtable

## Требования

- Node.js ≥ 18
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))
- Airtable API Key и Base ID

## Установка

1. Клонируйте репозиторий или скопируйте файлы проекта

2. Установите зависимости:
```bash
yarn install
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Заполните переменные окружения в файле `.env`:
```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
AIRTABLE_API_KEY=your_airtable_api_key
AIRTABLE_BASE_ID=your_base_id
```

## Настройка Airtable

В вашей базе Airtable должны быть следующие таблицы:

### Таблица "Ученики"
- `Telegram ID` (Number) - ID пользователя в Telegram
- `Текущий урок` (Link to "Уроки") - связь с текущим уроком

### Таблица "Задания"
- `Активно` (Checkbox) - активность задания
- `Урок` (Link to "Уроки") - связь с уроком
- `Текст задания` (Single line text) - текст задания
- `Правильные ответы` (Long text) - правильные ответы (каждый с новой строки)
- `Комментарий (верно)` (Single line text) - комментарий при правильном ответе
- `Комментарий (ошибка)` (Single line text) - комментарий при неправильном ответе
- `order` или `Порядок` (Number) - порядок задания в уроке

### Таблица "Прогресс"
- `Ученик` (Link to "Ученики") - связь с учеником
- `Задание` (Link to "Задания") - связь с заданием
- `Ответ ученика` (Single line text) - ответ ученика
- `Верно` (Checkbox) - правильность ответа

## Использование

### Регистрация

1. Ученик отправляет команду `/register` в Telegram
2. Бот показывает список доступных уроков
3. Ученик выбирает урок из списка
4. Бот автоматически создает запись в Airtable
5. Регистрация завершена!

### Начало урока

1. Ученик отправляет команду `/start` в Telegram
2. Бот проверяет регистрацию ученика
3. Если ученик найден, бот загружает задания текущего урока
4. Бот отправляет первое задание
5. Ученик отвечает на задание
6. Бот проверяет ответ и отправляет комментарий
7. При правильном ответе бот переходит к следующему заданию
8. При неправильном ответе ученик может попробовать еще раз
9. Прогресс сохраняется в Airtable

## Разработка

Проект использует ES Modules. Все модули экспортируют именованные функции.

## Обработка ошибок

Бот включает обработку ошибок на всех уровнях:
- Валидация переменных окружения при запуске
- Обработка ошибок API Airtable
- Обработка ошибок Telegram API
- Информативные сообщения об ошибках для пользователей
